# Боевой workflow
Не только инициатива, а полный цикл раунда:
объявил действие -> бросок -> урон -> soak -> применить раны -> лог.
Это одна из самых “ощущаемых” фич для хроник, где бои бывают редко, но каждый раз все вспоминают механику заново.

# Frenzy / Rötschreck / degeneration assistant
Очень “вампирская” штука. Когда персонаж получает триггер, система подсказывает:
какой бросок нужен, какая сложность типична, какие последствия возможны, и сохраняет это в лог.
Это сильно поддерживает атмосферу и разгружает ST.

# Лента хроники как полноценный session journal
Сейчас у вас уже есть лог событий. Его можно дорастить до “дневника хроники”:
сцены, итоги сессии, решения, обещания NPC, последствия, зацепки на следующую встречу.

# NPC roster для хроники
Не только персонажи игроков, а отдельный справочник NPC:
клан, секта, поколение, статус, отношение к котерии, долги, где последний раз появлялся.
Для VtM это почти обязательный инструмент.
